optionTile

jQuery that grabs all marked SELECT inputs on a page and transforms them automatically into tile-like option sets. Supports displaying an optional "option cost." This was inspired by a use case where the "options" effect of the Apple Store product configurator was desired, in an app that could only output SELECT menus for data.

MIT License.

optionTile was designed to be responsive, and to suggest the tiles of "flat interfaces," but you can style it any way you want.

To turn a SELECT input into an optionTile, just add the class .option-tile-source to the SELECT input.
